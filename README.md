# 剩余价值计算器 / Remaining Value Calculator

本项目是一个基于 Web 的剩余价值计算器，用户可以通过输入汇率、金额、周期等信息来计算剩余价值。此项目主要由 HTML、CSS、JavaScript 和 Node.js 构成。

## 环境要求

在开始之前，请确保您的开发环境满足以下要求：

- Node.js 版本 >= 12
- 浏览器支持最新的 HTML5 和 JavaScript
- 一个文本编辑器（推荐使用 VSCode）
